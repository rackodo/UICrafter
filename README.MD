# UICrafter

Mix and match resource packs to get your own customised pack.

## Goals

- Load resource packs as their individual GUI components.
- Allow the user to pick between components from the resource packs.
- (Optional) Offer customisation of output resource pack's name and description text.
- Allow the user to download the resulting resource pack as a .zip archive.
    - Include crediting of the authors of used resource packs.

## Included Packs

- [Default-Dark-Mode | nebulr](https://github.com/nebuIr/Default-Dark-Mode)
- [Faithful-32x-Java | Faithful Resource Pack](https://faithfulpack.net)
- [Immersive-Interfaces | Shrimpsnail](https://github.com/Shrimpsnail/Immersive-Interfaces)
